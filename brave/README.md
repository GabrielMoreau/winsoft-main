# Brave - Web browser

Brave is a free and open-source web browser. It is developed by US-based
Brave Software, Inc. and based on the Chromium web browser. Brave is
marketed as a privacy-focused web browser and includes features such as
built-in advertisement blocking, protections against browser
fingerprinting and a private browsing mode that integrates the Tor
anonymity network.
