# Optgeo - Simulating two-dimensional geometric optics

OptGeo is free software for simulating two-dimensional geometric optics.
It is open-source software distributed free of charge under the terms of the GNU General Public License.

* Website : http://jeanmarie.biansan.free.fr/optgeo.html
* Framalibre : https://framalibre.org/notices/optgeo.html

Uninstall silent command:

```bat
"C:\Program Files (x86)\Optgeo\unins000.exe" /VERYSILENT /NORESTART
```


## Register Key

Example :

 | Hive | DisplayName | Publisher | DisplayVersion | KeyProduct | UninstallExe |
