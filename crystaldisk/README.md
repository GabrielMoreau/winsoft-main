# CrystalDiskInfo - HDD/SSD utility software which supports a part of USB, RAID and NVMe. 

CrystalDiskInfo is free and open-source software (MIT license) designed to monitor the status of hard drives and SSDs.
It uses S.M.A.R.T. data to detect wear, errors, and failure risks.
The software displays a clear health status (good, caution, or bad).
It allows you to view the disk temperature in real time.
It also provides detailed technical information (hours of use, startups, interface).
Alerts can notify the user in case of a critical problem.

* Website : https://crystalmark.info/en/software/crystaldiskinfo/

* Download : https://crystalmark.info/en/download/#CrystalDiskInfo
* Silent install : `/VERYSILENT /NORESTART`

The package installs only the Standard version (there are four versions: Basic, Standard, Full, Mega).
