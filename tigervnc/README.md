# TigerVNC - Virtual Network Computing (VNC) client

TigerVNC is a free and open-source Virtual Network Computing (VNC)
server and client software, started as a fork of TightVNC in 2009.

TigerVNC has a different feature set than TightVNC, despite its origins.
For example, TigerVNC adds encryption for all supported operating
systems and not just Linux. Conversely, TightVNC has features that
TigerVNC doesn't have, such as file transfers.

TigerVNC focuses on performance and on remote display functionality.
