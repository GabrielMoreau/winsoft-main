# CrystalDiskMark - Simple disk benchmark software

CrystalDiskMark is free and open-source software (MIT license) designed to benchmark hard drives and SSDs.
It may shorten SSD/USB Memory life.
Benchmark result is NOT compatible between different major version.

* Website : https://crystalmark.info/en/software/crystaldiskmark/

* Download : https://crystalmark.info/en/download/#CrystalDiskMark
* Silent install : `/VERYSILENT /NORESTART`

The package installs only the Standard version (there are four versions: Basic, Standard, Full, Mega).


## Register Key

Example :

 | Hive | DisplayName | Publisher | DisplayVersion | KeyProduct | UninstallExe |
 |:---- |:----------- |:--------- |:-------------- |:---------- |:------------ |
 | HKLM | CrystalDiskMark 9.0.2 | Crystal Dew World | 9.0.2 | `CrystalDiskMark9_is1` | `"C:\Program Files\CrystalDiskMark9\unins000.exe"` |
